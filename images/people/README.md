# People photos

Put collaborator/student photos here, then reference them in `_data/people.yml`
as `photo: /images/people/<filename>`.

Tips:
- Square or portrait images work best (cards crop to a 4:5 box).
- ~600×750 px is plenty; keep files reasonably small (JPG/PNG/WebP).
- If a person has no `photo`, the card shows their first initial instead.
