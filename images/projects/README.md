# Project photos

Put one image per research agenda here, then reference it in
`_data/projects.yml` as `photo: /images/projects/<filename>`.

Tips:
- Landscape images work best (cards crop to a 4:3 box).
- ~1000×750 px is a good size; keep files reasonably small (JPG/PNG/WebP).
- If an agenda has no `photo`, the card simply shows a neutral placeholder box.
